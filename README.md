# デュエルマスターズ風カードゲーム

## 概要
デュエルマスターズをモチーフにした対戦型カードゲームです。

## 特徴
- 5つの文明（火・水・自然・光・闇）
- シールド5枚制
- マナシステム
- リアルタイム対戦

## 起動方法
```bash
python app.py
```

ブラウザで http://localhost:5001 にアクセス

## ルール
1. 各プレイヤーはシールド5枚でスタート
2. 毎ターン1枚ドロー
3. 手札からマナゾーンにカードをチャージ
4. マナを使ってクリーチャーを召喚
5. クリーチャーで攻撃してシールドを破壊
6. 相手のシールドを全て破壊したら勝利！
